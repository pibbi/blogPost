How to Scaffold a Project

This article explains how to create a project folder using the command line. Once you've completed these steps, you'll have a new desktop folder created and filled with important folders for web development projects and organization.

Before you begin, you should be comfortable with the command line interface and you should know how to do basic tasks from the terminal.

Important note: The terminal commands in the instructions below include the symbol $ to represent the command prompt. When entering the commands, you do not type the $, only the specific commands themselves.

Step 1: Open the terminal, and change directory to desktop.

Once in terminal, type the following command: 

$ cd desktop

Step 2: Create the main project folder.

Now you're reading to start your project folder! This will be the main folder that appears on the desktop, and inside will be all the other necessary project folders. I'm going to call this one "new-project", but you can name it whatever you'd like. Enter: 

$ mkdir new-project

Step 3: Enter the new file and create subfolders.

Now, you have to enter the new-project folder you just made in order to create the subfolders inside of it.

$ cd new-project

Create the new folders.

$ mkdir images scripts site styles index.html

Now you have your main folder called:

new-project
it's inner folders called: 
images
scripts
site
styles
and the html file:
index.html

Step 4: Fill thestylesfolder.

Enter the styles folder.

$ cd styles

Create a css file within this folder for styling.

$ touch style.css

Step 5: Fill thesitefolder.

Change your directory back to thenew-projectfolder. 

$ cd ..

Enter thesitefolder. 

$ cd site

Fill the folder with it's subfolders.

$ mkdir comments pages posts

That's it!

Now, if you check your desktop you should have a new folder there titled accordingly, full of the folders you created.
